Comment fonctionne ce jeu ?

1.Démarrage

Cliquez sur "Démarrer le jeu" pour lancer la partie. Vous aurez 30 secondes pour attraper un maximum de taupes

2.Attraper les taupes

Les taupes apparaissent aléatoirement dans les trous. Cliquez dessus dès qu'elles apparaissent pour marquer des points.

3.Score

Chaque taupe attrapée ajoute un point à votre score. Le score est affiché en haut de l'écran.

4.Fin du jeu

Lorsque le temps est écoulé,le jeu se termine. Vous pouvez relancer le jeu en cliquant sur "Relancer le jeu".

Amusez-vous bien !
